Environment Dynamics Matters: Embodied Navigation to Movable Objects (EDM)


IROS 2024 Workshop

Introduction

The object-goal navigation is extensively studied in the Embodied-AI community, while they are mostly limited to static objects where scene graph is often used to learn the association between the static object and environment. In reality, it is definitely more likely that the objects we want to find are movable or have some temporal location patterns associated with user routines. In recent years, a series of research work considered the dynamics of object movement rather than searching static objects. Coincidently, the AIGC technology is developing rapidly and becoming mature, which make it possible for us to generate appropriate simulation environment for the research of dynamic embodied navigation. We believe it might be just a perfect time to gather researchers who are interested in this topic to have a discussion together.

The tentative content of the workshop includes the following aspects:
  • extraction of spatio-temporal object information from the long-term interaction with the environment
  • modeling of spatio-temporal object information using advanced machine learning technologies such as graphical model
  • generation of dynamic simulation environment using AIGC technologies
