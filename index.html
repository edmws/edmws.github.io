Environment Dynamics Matters:
Embodied Navigation to Movable Objects


IROS 2024 Tentative Workshop

Introduction

The semantic object-goal navigation, often inspired by the cognitive mechanisms, is extensively studied in the Embodied-AI community, while they are mostly limited to static objects where scene graph is often used to learn the association between the static object and environment. In reality, it is definitely more likely that the objects we want to find are movable or have some temporal location patterns associated with user routines. In recent years, a series of research work considered the dynamics of object movement rather than searching static objects. Coincidently, the AIGC technology is developing rapidly and becoming mature, which make it possible for us to generate appropriate simulation environment for the research of dynamic embodied navigation and navigation. Therefore, we believe it might be just a perfect time to gather researchers to have discussions together and further accelerate the development of the field.

This workshop aims to bring researchers from both the academia and industry to discuss the importance of environment dynamics in the embodied navigation and manipulation task. The tentative content of the workshop includes the following aspects:
  • extraction of spatio-temporal object information from the long-term interaction with the environment
  • modeling of spatial-temporal object information using advanced machine learning technologies such as graphical model and or cognitive-inspired method
  • generation of dynamic simulation environment using AIGC technologies (LLM, diffusion, and so on)
  • planning of smart actions for dynamic embodied navigation and manipulation
  • deployment of dynamic embodied navigation and manipulation on real-world robotic platform
